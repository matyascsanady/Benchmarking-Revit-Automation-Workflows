# Benchmarking-Revit-Automation-Workflows

This repository accompanies the MSc thesis:

_“Benchmarking Revit Automation Workflows: A Quantitative and Qualitative Evaluation”_

Budapest University of Technology and Economics, 2025

## Description

The repository contains reference implementations of two benchmark automation tasks in Autodesk Revit, developed under controlled conditions to support quantitative and qualitative evaluation of automation workflows.

## Reproducibility

The release `v1.0-thesis-submission` corresponds exactly to the implementations evaluated in the thesis.
All tests were performed in Autodesk Revit 2026.
